## 1. 현재 발생한 주요 문제점 및 긴급 수정 사항 (우선순위 1)

최근 구현에서 아래와 같은 치명적인 UI 결함이 발견되었습니다. 코드를 즉각 수정해야 합니다.

### 1.1 FloatingAiCard 위치 및 레이아웃 오류
- **문제**: 카드가 좌측 상단 등 엉뚱한 위치에 고정되거나, 너무 둥근 모양(타원형)으로 렌더링되며, 첫 화면 진입 시 펼쳐진 채로 시작해 화면을 가립니다.
- **해결 요구사항**:
  - `FloatingAiCard.module.css`에서 `bottom`과 `left`, `transform`을 조정해 **화면 하단 중앙**에 안정적으로 배치되도록 수정하십시오.
  - `border-radius`를 지나치게 둥근 값이 아닌 적당한 사각형(`var(--radius-lg)` 수준)으로 조정하십시오.
  - 컴포넌트 초기 상태를 `expanded = false`로 변경하여 첫 진입 시 요약만 보이게 하십시오.

### 1.2 FloatingAiCard 텍스트 잘림 현상
- **문제**: "신뢰도: 보" 처럼 글씨가 중간에 잘리거나, 설명 텍스트가 컨테이너를 벗어나거나 숨겨지는 문제가 있습니다.
- **해결 요구사항**:
  - `footer`나 `newsItem` 등 텍스트를 담는 요소의 CSS에 `white-space: nowrap`으로 인한 잘림 현상을 해결하십시오. (`word-break: keep-all`, `white-space: normal` 활용)
  - `flex` 레이아웃에서 텍스트가 줄어들지 않도록 `flex: 1` 및 최소 너비 설정을 보강하십시오.

### 1.3 우측 상단 플로팅 버튼 겹침 현상
- **문제**: 학습 모드 토글 버튼과 포트폴리오 샌드박스 진입 버튼이 우측 상단에서 서로 겹쳐서 표시됩니다.
- **해결 요구사항**:
  - `App.jsx`와 `App.module.css`에서 이 두 버튼을 별도의 `absolute` 위치로 각각 두지 말고, **Flexbox 컬럼 기반의 `rightActionGroup` 컨테이너**로 묶어 간격(`gap`)을 주어 세로로 깔끔하게 정렬하십시오.

### 1.4 PortfolioSandbox 내부 UI 붕괴
- **문제**: 샌드박스를 열었을 때 내용이 잘려보이고, 글씨가 뒤섞여 보이며, 패널 하단이 완전히 공란으로 비어있어 미관상 매우 좋지 않습니다.
- **해결 요구사항**:
  - `PortfolioSandbox.module.css`에서 콘텐츠 영역을 `display: flex; flex-direction: column; flex: 1;` 구조로 변경하여 패널 높이를 꽉 채우도록 구성하십시오.
  - 하단의 AI 리뷰 섹션(`aiReviewSection`)에 `margin-top: auto`를 적용하여 패널의 가장 아래쪽으로 밀어내 빈 공간을 자연스럽게 채우십시오.
  - 아이콘과 텍스트가 겹치지 않도록 `reviewCard p` 요소에 `flex: 1`을 적용하고 텍스트 줄바꿈을 허용하십시오.

## 2. 현재 문제

사용자는 현재 결과에 만족하지 않는다.

이유:

- 화면이 여전히 크게 바뀐 것처럼 보이지 않는다.
- 기존 브리프/운영 도구 느낌이 남아 있다.
- 첫 화면에 정보가 너무 많아 헷갈린다.
- 사용자가 무엇을 해야 하는지 즉시 알기 어렵다.
- 주식 용어와 개념 학습 내용이 너무 적고 얕다.
- 그래프를 보며 실제로 공부되는 느낌이 부족하다.
- 어디에 투자 후보를 둘지, 왜 봐야 하는지, 매수/매도 검토 구간이 충분히 직관적이지 않다.
- AI가 제품 핵심으로 제대로 활용된다는 느낌이 약하다.

## 4. 프론트 전면 재설계 지시

### 4.1 기존 프론트를 없다고 생각하라

기존 `App.jsx`, `AppSections.jsx`, CSS 구조를 그대로 살리는 방식으로 접근하지 마라.

필요하면 아래를 전부 새로 설계한다.

- App shell
- Header
- First screen
- Navigation
- Search
- Chart workspace
- AI insight panel
- Learning panel
- Portfolio panel
- Admin area
- CSS token
- Layout system
- Component structure

단, 기존 백엔드 API와 기존 기능은 삭제하지 않는다. 일반 사용자 화면에서 안 보이게 정리하고, 관리자/기록 영역으로 분리한다.

### 4.2 첫 화면은 정보 과부하 금지

첫 화면에서 너무 많은 정보를 보여주지 마라.

첫 화면에 반드시 보일 것:

1. 통합 검색
2. 메인 차트
3. AI 핵심 인사이트
4. 오늘 볼 만한 관심 후보 3개 이하
5. 초보자 학습 진입점

첫 화면에서 숨기거나 접을 것:

- 긴 브리프 전문
- 검증 상세
- 수집 노트
- 관리자 버튼
- 백필/생성/보관
- 긴 TOP 리스트
- 외부 링크 반복
- 과도한 근거 원문

첫 화면 목적:

사용자가 5초 안에 `검색하거나`, `차트를 보거나`, `AI에게 물어보면 된다`를 이해해야 한다.

### 4.3 그래프가 메인이다

사용자 입장에서는 그래프를 기준으로 실제 공부가 되어야 한다.

메인 차트 요구사항:

- 일봉/주봉/월봉 전환
- 가격 흐름
- 거래량
- 이동평균선
- 이벤트 마커
- AI 해석 마커
- 매수 검토 구간
- 분할매수 검토 구간
- 관망 구간
- 매도 검토 구간
- 리스크 관리 구간
- hover tooltip
- 모바일에서도 조작 가능

차트 tooltip에는 아래가 들어가야 한다.

- 이 구간의 의미
- 초보자 설명
- 근거
- 반대 신호
- 리스크
- 데이터 기준일
- 신뢰도
- 관련 용어 링크

### 4.4 투자 후보와 매수/매도 UX

사용자는 `어디에 투자해야 하는지`, `왜 봐야 하는지`, `언제 매수/매도 검토를 해야 하는지` 알고 싶어 한다.

단, 법적/신뢰성 리스크 때문에 무조건적 지시는 금지한다.

금지 표현:

- 지금 사라
- 지금 팔아라
- 반드시 오른다
- 수익 보장
- 무조건 추천

허용 표현:

- 관심 후보
- 살펴볼 만한 이유
- 매수 검토 구간
- 분할매수 검토 구간
- 관망 구간
- 매도 검토 구간
- 리스크 관리 구간
- 이 조건이 확인되면 검토
- 이 반대 신호가 나오면 보류

각 후보 종목에는 아래를 보여준다.

- 왜 관심 후보인가
- 관련 산업/테마
- 차트 상태
- 매수 검토 조건
- 매도 검토 조건
- 관망해야 하는 조건
- 리스크
- 초보자가 확인할 체크리스트
- AI 근거 요약

### 4.5 학습 탭을 완전히 강화하라

학습은 부가 기능이 아니다. 초보자 retention의 핵심이다.

7. 관련 차트 구간
8. 관련 질문
9. AI에게 물어보기 버튼

## 5. AI 활용 방식

AI는 제대로 활용하되, 사고 과정을 그대로 노출하지 않는다.

절대 금지:

- AI의 내부 사고 과정 그대로 보여주기
- chain-of-thought 노출
- "제가 단계별로 생각해보면..." 같은 내부 추론 문장
- 근거 없는 확정 표현

대신 사용자에게 보여줄 것:

- 한 줄 결론
- 핵심 근거
- 반대 신호
- 리스크
- 초보자 설명
- 데이터 기준일
- 신뢰도
- 한계
- 출처
- 다음 확인 체크리스트

AI 기능은 화면에서 명확해야 한다.

필수 AI 영역:

- AI 시장 요약
- AI 차트 해석
- AI 매수/매도 검토 구간 설명
- AI 리스크 요약
- AI 용어 설명
- AI 포트폴리오 점검

AI는 사용자가 이해할 수 있는 결과를 보여주는 도구다. AI 자체를 과시하지 말고, 사용자가 더 잘 판단하도록 돕는다.
